### Contract 7: 原子寫入策略

**規則**: 避免並行生成競態

```python
def save_report_atomic(self, content: str, target_path: Path) -> Path:
    """temp → flush → fsync → os.replace"""
    temp_path = target_path.with_suffix(f".tmp.{uuid4().hex[:8]}")
    try:
        with open(temp_path, "w", encoding="utf-8") as f:
            f.write(content)
            f.flush()
            os.fsync(f.fileno())
        os.replace(temp_path, target_path)
        fsync_directory(target_path.parent)  # POSIX only
        return target_path
    except Exception:
        temp_path.unlink(missing_ok=True)
        raise

def fsync_directory(dir_path: Path) -> None:
    """POSIX: fsync(dir_fd), Windows: no-op"""
    if sys.platform == "win32":
        return
    dir_fd = os.open(str(dir_path), os.O_RDONLY | os.O_DIRECTORY)
    try:
        os.fsync(dir_fd)
    finally:
        os.close(dir_fd)
```

### Contract 8: Rebuild 整合

**規則**: `lc rebuild --target reports` 使用相同生成器

```python
def rebuild_reports(data_dir: Path, force: bool = True) -> None:
    generator = ReportGenerator(data_dir)
    generator.generate_all(
        projection=load_projection(data_dir),
        comparison=load_comparison(data_dir),
        format="md",
        save=True,
        force=True,  # 忽略 cache
    )
```

### Contract 9: Error Contract

**Exit Codes**:

| Code | 意義 | 情境 |
|------|------|------|
| 0 | 成功 | 所有報表生成成功 |
| 1 | 一般錯誤 | 未分類 |
| 2 | 輸入缺失 | Phase 2 輸出不存在 |
| 3 | 渲染錯誤 | 模板/格式化失敗 |
| 4 | 快取錯誤 | cache read/write 失敗 |
| 5 | I/O 錯誤 | 磁碟空間/權限問題 |

**輸入依賴行為**:

| projection | comparison | 行為 |
|------------|------------|------|
| ✅ | ✅ | 生成全部 3 報表 |
| ✅ | ❌ | 生成 2 報表，跳過 scenario_comparison |
| ❌ | ✅ | Exit 2（projection 必要） |
| ❌ | ❌ | Exit 2 |

## §4 核心 API 設計

### ReportGenerator 類

```python
class ReportGenerator:
    def __init__(self, data_dir: Path): ...

    def generate_monthly_summary(
        self, projection: ProjectionResult, format: str = "md"
    ) -> ReportOutput:
        """月度現金流摘要：期間總收入/支出、平均月現金流、赤字月數"""

    def generate_projection_table(
        self, projection: ProjectionResult, format: str = "md"
    ) -> ReportOutput:
        """12-24 月預測表：每月收入/支出/淨現金流/累積儲蓄"""

    def generate_scenario_comparison(
        self, comparison: ScenarioComparisonResult, format: str = "md"
    ) -> ReportOutput:
        """情境比較表：各情境假設、最終儲蓄、vs 基準差異"""

    def generate_all(
        self,
        projection: ProjectionResult,
        comparison: Optional[ScenarioComparisonResult],
        format: str = "md",
        save: bool = False,
    ) -> list[ReportOutput]:
        """生成所有報表"""
```

### ReportOutput 模型

```python
@dataclass(frozen=True)
class ReportOutput:
    report_type: str              # monthly_summary | projection_table | scenario_comparison
    content: str                  # Markdown 或 JSON 內容
    format: str                   # md | json
    provenance: ReportProvenance

    def save(self, output_dir: Path) -> Path:
        """存檔到指定目錄"""
```

## §5 CLI 設計

### 基本用法

```bash
lc report                         # 所有報表 → stdout (md)
lc report --type monthly          # 只生成月度摘要
lc report --format json           # JSON 格式
lc report --format md,json --save # 多格式（須 --save）
lc report --save                  # 存檔到 derived/reports/

lc report --allow-missing         # 缺少輸入時跳過（預設報錯）
lc report --force                 # 強制重新生成
```
